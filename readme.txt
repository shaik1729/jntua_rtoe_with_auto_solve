# create your own account of noaichaptcha api service

1. goto the below url

    https://dash.nocaptchaai.com/

2. click on signup and fill the details

# steps for running:

1. delete the env folder

2. create python environment 

    python3 -m venv env
    or
    py -m venv env

3. install the dependences

    pip3 install -r requirements.txt

4. run the main.py

    python3 main.py
    or
    py main.py

5. enter the username and password of the noai 

6. enter the result url

7. submit
